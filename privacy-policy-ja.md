## 2. 本プラグインが収集・送信する情報

### 2.1 ライセンスの確認（インターネット経由）

本プラグインの Pro 機能の利用にあたり、**ライセンスキー**を運営者が指定するサーバー（以下「ライセンスサーバー」）へ送信し、有効性を確認する場合があります。

あわせて、Figma の環境で利用可能な範囲で、**Figma のユーザー識別子**（プラグイン API が提供するユーザー ID 等）をライセンスサーバーへ送信する場合があります。送信の有無・内容は、本プラグインおよびライセンスサーバーの実装に従います。

**ライセンス検証に利用するホスト**は次のとおりです。

- `https://rsp-vercel-license.vercel.app`（ライセンス検証用 API）

通信は HTTPS を想定しています。

### 2.2 端末内に保存する情報（Figma / プラグインのストレージ）

本プラグインは、Figma が提供する**クライアント側ストレージ**に、ライセンスキーや設定値（例: 塗り色や余白の段階など）を保存する場合があります。**これらは主に利用者の端末（Figma クライアント環境）側に留まる**設計であり、運営者が任意にリモートから当該データにアクセスするものではありません。ただし、上記 2.1 の通り、ライセンス確認のためにキーがサーバーへ送信される場合があります。

### 2.3 デザインファイルの内容

本プラグインは、**利用者が Figma 上で選択・操作するオブジェクト**にアクセスします。これはプラグイン機能（リサイズ、色の変更、レイアウト調整など）を提供するために必要な範囲に限られます。**デザインファイルの内容を、上記のライセンス確認以外の目的で運営者のサーバーにアップロードすることはありません**（プラグインの実装がそうなっている限り）。
